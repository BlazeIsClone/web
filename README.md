# Blaze WEB Monorepo

This project is bootstraped with Payload CMS and NextJS.

# Links

Fronend: `http://localhost:3000`

CMS Admin: `http://localhost:3000/admin`

# Commands

Run in developmnet application mode.

```bash
npm start
```

<br/>

Build static files.

```bash
npm run build
```

<br/>

Destroy can clear container data.

```bash
npm run down
```
